## dog fight

```

```

Sound effect from Home World: Desert of Kharak.

Sound effect extracted by Lev_Astov from reddit.

### Control:

| Player | Direction  | Gatlin  | Missle |
| ------ | ---------- | ------- | ------ |
| 0      | A/D        | L_Shift | L_Ctrl |
| 1      | Left/Right | R_Shift | R_Ctrl |
